# Usage
نحوه ی استفاده از کتابخانه :

1- کد زیر را به فایل گردل اصلی پروژه اضافه کنید.
```javascript
	allprojects {
		repositories {
			...
			maven { url "https://jitpack.io" }
		}
	}
```
2- کد زیر را به فایل گردل ماژول خود اضافه کنید.
```javascript
	dependencies {
    	compile 'com.abbas-oveissi:materialsearchview:1.0.3'
	}
```
